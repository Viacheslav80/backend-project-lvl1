# Brain-games clone project.

## installing
 
build the pocket.

```
$ uses_folder/make publish 
```
install pocket of global

```
$ uses_folder/sudo npm link
```

use:

```
$ brain-even
$ brain-calc
$ brain-gcd
$ brain-prime
$ brain-prograssion
```
